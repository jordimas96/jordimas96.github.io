Principal desenvolupador d'una web per al Ministeri d'Exteriors

En aquesta empresa vaig treballar per a Indra, una consultora que gestiona diversos projectes per al Gobierno de España, contractat per Alvea.
Principal desenvolupador frontend d'un backoffice per al Ministerio de Asuntos Exteriores y Cooperación, que permetrà als consolats i ambaixades proporcionar suport als ciutadans espanyols que es troben a l'estranger durant situacions de crisi, com ara atemptats terroristes o catàstrofes naturals. L'aplicació permetrà gestionar grups de ciutadans per a organitzar trasllats, salconduits, repatriacions, renovació de documents, comunicacions i atenció a necessitats especials, entre d'altres serveis.

Vam iniciar el projecte creant una llibreria de components en Angular, utilitzant PrimeNG, Angular Material i altres plugins com a base. Aquesta llibreria inclou tant elements comuns en formularis (selectors, radiobuttons, checkboxes, calendaris, mapes interactius) com components habituals en interfícies d'usuari (botons, cards, toasts, menús desplegables, breadcrumbs, etc.).

Tant els components com la web estan desenvolupats seguint uns estils estrictament definits, amb cada aspecte del disseny (colors, formes, espais, fonts, mides, animacions) fortament marcat. L'estructura de la pàgina és semblant a la de les altres webs oficials de l'estat. Els estils estan completament desacoblats i organitzats.

L'aplicació compta amb suport per a diversos idiomes: castellà, anglès, català, euskera, gallec, asturià i francès.

L'aplicació és una Single Page Application d'Angular. Té sistema d'emmagatzematge en caché de les peticions per a garantir-ne el funcionament en zones amb poca cobertura, o amb problemes de connexió, cosa freqüent en situacions d'emergència. A més, compleix tots els estàndards d'accessibilitat de nivell AA.

Assoliments

  • Participació en el desenvolupament i maquetació de la nostra pròpia llibreria completa de components per al projecte i també per a altres webs oficials del Ministeri
  • Vaig desenvolupar diversos components per a un projecte específic que van agradar tant que, sense haver sigut dissenyats per a la llibreria, van acabar integrant-s'hi i van ser adoptats en diverses aplicacions oficials del Ministeri
  • Desenvolupament de l'aplicació seguint una guia d'estils molt estricta
  • En poc temps em vaig erigir com el desenvolupador de referència en CSS i maquetació dins l'equip

Principal desarrollador de una web para el Ministerio de Exteriores

En esta empresa trabajé para Indra, una consultora que gestiona varios proyectos para el Gobierno de España, contratado por Alvea.
Principal desarrollador frontend de un backoffice para el Ministerio de Asuntos Exteriores y Cooperación, que permitirá a los consulados y embajadas proporcionar soporte a los ciudadanos españoles que se encuentran en el extranjero durante situaciones de crisis, como atentados terroristas o catástrofes naturales. La aplicación permitirá gestionar grupos de ciudadanos para organizar traslados, salvoconductos, repatriaciones, renovación de documentos, comunicaciones y atención a necesidades especiales, entre otros servicios.

Iniciamos el proyecto creando una librería de componentes en Angular, utilizando PrimeNG, Angular Material y otros plugins como base. Esta librería incluye tanto elementos comunes en formularios (selectores, radiobuttons, checkboxes, calendarios, mapas interactivos) como componentes habituales en interfaces de usuario (botones, cards, toasts, menús desplegables, breadcrumbs, etc.).

Tanto los componentes como la web están desarrollados siguiendo unos estilos estrictamente definidos, con cada aspecto del diseño (colores, formas, espacios, fuentes, tamaños, animaciones) fuertemente marcado. La estructura de la página es similar a la de las otras webs oficiales del estado. Los estilos están completamente desacoplados y organizados.

La aplicación cuenta con soporte para varios idiomas: español, inglés, catalán, euskera, gallego, asturiano y francés.

La aplicación es una Single Page Application de Angular. Tiene sistema de almacenamiento en caché de las peticiones para garantizar su funcionamiento en zonas con poca cobertura, o con problemas de conexión, algo frecuente en situaciones de emergencia. Además, cumple con todos los estándares de accesibilidad de nivel AA.

Lead developer of a website for the Ministry of Foreign Affairs

In this company I worked for Indra, a consultancy that manages several projects for the Government of Spain, hired by Alvea.
Lead frontend developer of a back-office system for the Ministry of Foreign Affairs and Cooperation, which will allow consulates and embassies to provide support to Spanish citizens who are abroad during crises situations, such as terrorist attacks or natural disasters. The application will allow managing groups of citizens to organize transfers, safe conduct passes, repatriations, document renewals, communications and attention to special needs, among other services.

We started the project by creating a component library in Angular, using PrimeNG, Angular Material and other plugins as a base. This library includes both common form elements (selectors, radio buttons, checkboxes, calendars, interactive maps) and common user interface components (buttons, cards, toasts, drop-down menus, breadcrumbs, etc.).

Both the components and the website are developed following strictly defined styles, with each aspect of the design (colors, shapes, spaces, fonts, sizes, animations) strongly established. The structure of the page is similar to that of the other official government websites. The styles are completely decoupled and organized.

The application has support for several languages: Spanish, English, Catalan, Basque, Galician, Asturian and French.

The application is an Angular Single Page Application. It has a request caching system to ensure its operation in areas with poor coverage, or with connection problems, something common in emergency situations. It also meets all level AA accessibility standards.

Achievements

  • Participation in the development and layout of our own complete library of components for the project and also for other official websites of the Ministry
  • I developed several components for a specific project which were so liked that, without having been designed for the library, they ended up being integrated there and were adopted in many of the Ministry's official applications
  • Application development following a very strict style guide
  • In a short time I established myself as the reference developer for CSS and layout within the team
