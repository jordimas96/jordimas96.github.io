# IN2ART – Art social network

IN2ART is a social network for artists, collectors, and art enthusiasts. The platform is a combination of social interaction, artwork discovery, and artistic content management.

## Main features

### Authentication and user profile

* Secure login with multilingual support.
* Customizable profile with biography, email, links, and user type: professional artist, collector, or viewer.
* Viewing followers, subscribers, and list of own posts (separated by type of artwork).
* Most popular posts highlighted on the profile.

### Post feed

* Infinite scroll with dynamic content loading.
* Suggested posts in the infinite scroll come from 3 different sources:
  * Followed users
  * Recommended content based on interests
  * Featured artworks
* Each post can include multiple audiovisual files (images, videos, audios), with plans to add support for interactive content via iframes.
* Full interaction: voting by medium or style, comments, replies, and comment voting.

### Post creation and management

* Intuitive form to upload new artworks:
  * Audiovisual files (with the ability to reorder by dragging)
  * Title, own or recognized author, date (exact or approximate)
  * Medium, artistic movement, technique, dimensions, price, description, and tags
  * Age-restricted content control
  * Post preview before uploading
* Forms designed for maximum usability and ergonomics: drag to reorder audiovisual files, flexible date selectors, automatic tag management.

### Search and exploration

* Global search for artworks, users, categories, tags, movements, and media.
* Personalized exploration function based on interests and user type when no active search.
* Quick access from posts to searches by tags or categories.

### Notifications and settings

* Notification system for interactions and updates.
* Full settings section to manage preferences and account configuration, as well as options to deactivate or delete it.
