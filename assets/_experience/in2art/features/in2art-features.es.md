# IN2ART – Red social de Arte

IN2ART es una red social enfocada en artistas, coleccionistas y aficionados al arte. La plataforma es una combinación de interacción social, descubrimiento de obras y gestión de contenido artístico.

## Funcionalidades principales

### Autenticación y perfil de usuario

* Login seguro con soporte multilenguaje.
* Perfil personalizable con biografía, correo, enlaces y tipo de usuario: artista profesional, coleccionista o espectador.
* Visualización de seguidores, suscriptores y lista de publicaciones propias (separadas por tipo de obra).
* Publicaciones más populares destacadas en el perfil.

### Feed de publicaciones

* Scroll infinito con carga dinámica de contenido.

* Las publicaciones sugeridas en el scroll infinito vienen de 3 orígenes distintos:
  - Usuarios seguidos
  - Contenido recomendado según intereses
  - Obras destacadas

* Cada publicación puede incluir múltiples archivos audiovisuales (imágenes, vídeos, audios) y con planes de añadir soporte para contenido interactivo mediante iFrames.

* Interacción completa: votación según criterios del medio o estilo, comentarios, respuestas y votación de comentarios.

### Creación y gestión de publicaciones

* Formulario intuitivo para subir nuevas obras:

  * Archivos audiovisuales (con posibilidad de reordenar arrastrándolas)
  * Título, autor propio o reconocido, fecha (exacta o aproximada)
  * Medio, movimiento artístico, técnica, dimensiones, precio, descripción y tags
  * Control de contenido para mayores de edad
  * Previsualización de publicación antes de subrirla

* Formularios diseñados para máxima usabilidad y ergonomía: arrastrar para reordenar archivos audiovisuales, selectores de fecha flexibles, gestión de tags automática.

### Buscador y exploración

* Buscador global de obras, usuarios, categorías, tags, movimientos y medios.
* Función de exploración personalizada basada en intereses y tipo de usuario cuando no hay búsqueda activa.
* Acceso rápido desde publicaciones a búsquedas por tags o categorías.

### Notificaciones y ajustes

* Sistema de notificaciones de interacciones y novedades.
* Sección de ajustes completa para gestionar preferencias y configuración de la cuenta, así como opciones para desactivarla o borrarla.
